Western Digital My Passport Icons


This set of icons is created for the Western Digital My Passport line of products. The set contains icons that are compatible with Mac, Linux & Windows. The Mac versions of the icons are provided in ICNS and 'Get Info' copy & paste formats. The Linux version are provided in PNG format. The Windows version are provided in ICO format.


Icons for Mac & Linux are 512x512 in size, Windows icons are 256x256.
